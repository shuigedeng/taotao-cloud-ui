# 开发

## 本地开发

```shell
$ git clone https://github.com/Hellager/mission-backup.git
$ cd ./mission-backup
$ pnpm install
$ pnpm tauri dev
```

## 查看文档

```shell
$ cd ./mission-backup
$ pnpm docs:dev
```
